# Additional Statistics Topics — Regression, ANOVA, T-Tests, Confidence Intervals, Chi-Square, ML Basics

These extend file `01`'s theory coverage into a few more named tests/concepts that sometimes show up in entry-test theory MCQs. Same format — easy, plain English, answer right after each question.

## A. Regression

**Q1.** Regression analysis is primarily used to:
A) Measure the spread of a single dataset  B) Study the relationship between one variable and another, and predict outcomes  C) Sort data into categories  D) Calculate factorials
**Answer: B)**

**Q2.** In the simple linear regression equation y = a + bx, what does **b** represent?
A) The starting point (intercept)  B) The slope — how much y changes for each unit change in x  C) The total number of data points  D) The probability of an event
**Answer: B) The slope**

**Q3.** In y = a + bx, what does **a** (the intercept) represent?
A) The value of y when x = 0  B) The value of x when y = 0  C) The correlation coefficient  D) The standard deviation
**Answer: A)**

## B. ANOVA (Analysis of Variance)

**Q4.** ANOVA is primarily used to:
A) Compare the means of two or more groups at once  B) Calculate a single average  C) Measure correlation between two variables only  D) Find the mode of a dataset
**Answer: A)**

**Q5.** If an ANOVA test gives a very small p-value, it suggests:
A) All group means are equal  B) At least one group mean is significantly different from the others  C) The sample size was too small  D) The data is not normally distributed
**Answer: B)**

## C. T-Tests

**Q6.** A T-Test is typically used to:
A) Compare the means of two groups  B) Compare more than 5 groups at once  C) Measure the spread of one dataset only  D) Calculate the probability of independent events
**Answer: A)**

**Q7.** A "paired T-test" is used when:
A) Two completely unrelated, independent groups are compared  B) The same group is measured twice (e.g. before and after a treatment)  C) More than two groups are compared at once  D) No groups are being compared
**Answer: B)**

## D. Confidence Intervals

**Q8.** A 95% confidence interval means:
A) There's a 95% chance the true population value equals exactly the sample value  B) We are 95% confident the true population parameter lies somewhere within this calculated range  C) 95% of all data points fall inside this range, always  D) The sample size must be exactly 95
**Answer: B)**

**Q9.** As sample size increases (everything else constant), a confidence interval generally becomes:
A) Wider  B) Narrower (more precise)  C) Always exactly the same width  D) Impossible to calculate
**Answer: B) Narrower**

## E. Chi-Square Tests

**Q10.** A Chi-Square test is commonly used to:
A) Compare the means of continuous data  B) Test the relationship/independence between categorical variables  C) Calculate standard deviation  D) Find the median of a dataset
**Answer: B)**

## F. Hypothesis Testing — Type I / Type II Errors

**Q11.** A **Type I error** means:
A) Incorrectly rejecting a null hypothesis that was actually true (a "false positive")  B) Incorrectly accepting a null hypothesis that was actually false  C) Using the wrong formula  D) Having too small a sample size
**Answer: A)**

**Q12.** A **Type II error** means:
A) Incorrectly rejecting a true null hypothesis  B) Incorrectly failing to reject a null hypothesis that was actually false (a "false negative")  C) Choosing the wrong significance level  D) Using a biased sample
**Answer: B)**

**Q13.** The "significance level" (commonly written as α, e.g. 0.05) represents:
A) The probability of a Type I error that the researcher is willing to accept  B) The exact probability that the hypothesis is true  C) The required sample size  D) The correlation coefficient
**Answer: A)**

## G. Data Visualization Basics

**Q14.** Which chart best shows the distribution/frequency of a single continuous variable (e.g. exam scores)?
A) Pie chart  B) Histogram  C) Line chart only  D) Scatter plot only
**Answer: B) Histogram**

**Q15.** Which chart best shows the relationship between two continuous variables (e.g. height vs. weight)?
A) Bar chart  B) Pie chart  C) Scatter plot  D) Histogram
**Answer: C) Scatter plot**

**Q16.** A "box plot" (box-and-whisker plot) is mainly used to show:
A) Exact individual data values only  B) The spread, median, and possible outliers of a dataset  C) Correlation between two variables  D) Frequency counts only
**Answer: B)**

## H. Controlled Experiments

**Q17.** In a controlled experiment, the "control group" is:
A) The group that receives the treatment being tested  B) The group that does NOT receive the treatment, used as a baseline  C) Always the larger of the two groups  D) Not necessary for a valid experiment
**Answer: B)**

**Q18.** "Randomization" (randomly assigning subjects to groups) matters because it:
A) Increases the budget needed  B) Helps ensure the groups are similar on average, reducing bias  C) Removes the need for a control group  D) Guarantees a statistically significant result
**Answer: B)**

## I. Basic Machine Learning Concepts

**Q19.** "Supervised learning" means the model is trained on:
A) Data with no labels at all  B) Labeled data, where the correct output is already known  C) Only images  D) Only numerical data, never text
**Answer: B)**

**Q20.** "Unsupervised learning" means the model:
A) Is trained on labeled data with known outputs  B) Finds patterns or groupings in data without labeled outputs  C) Requires human supervision at every single step  D) Can only be used for regression
**Answer: B)**

---

## Quick-recall summary

| Test/Concept | One-line takeaway |
|---|---|
| Regression | Predicts one variable from another; slope = rate of change |
| ANOVA | Compares means across 3+ groups |
| T-Test | Compares means of 2 groups (paired = same group, twice) |
| Confidence Interval | A range we're X% confident contains the true value; bigger sample → narrower range |
| Chi-Square | Tests relationships between categorical variables |
| Type I error | False positive — rejected a true null hypothesis |
| Type II error | False negative — failed to reject a false null hypothesis |
| Histogram | Distribution of one variable |
| Scatter plot | Relationship between two variables |
| Box plot | Spread, median, outliers |
| Control group | The no-treatment baseline group |
| Supervised vs Unsupervised ML | Labeled data vs no labels |
